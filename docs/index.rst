Welcome to asyncpixel's documentation!
======================================

Asyncpixel is a modern, easy to use, feature-rich, and async ready API wrapper
for Hypixel.

**Features:**

- Modern Pythonic API using ``async``\/``await`` syntax
- Easy to use with an object oriented design
